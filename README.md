# Plagiarism Detection System

A research-based plagiarism detection system using Sentence-BERT for semantic similarity analysis.

## Features
- Semantic text similarity using Sentence-BERT
- Document upload and plagiarism checking
- Web-based interface
- Research-backed accuracy
